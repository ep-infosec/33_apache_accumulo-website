Plugin Documentation

Goals available for this plugin:

Goal Description
accumulo2:help Display help information on accumulo2-maven-plugin.
Call mvn accumulo2:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
accumulo2:start Goal which starts an instance of MiniAccumuloCluster.
accumulo2:stop Goal which stops all instances of MiniAccumuloCluster started with the start mojo.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.3.9
JDK 1.8
Memory No minimum requirement.
Disk Space No minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.apache.accumulo</groupId>
          <artifactId>accumulo2-maven-plugin</artifactId>
          <version>1.0.0</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.apache.accumulo</groupId>
        <artifactId>accumulo2-maven-plugin</artifactId>
        <version>1.0.0</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
